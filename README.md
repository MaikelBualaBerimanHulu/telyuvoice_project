# TelyuVoice

Ini adalah aplikasi yang membantu para mahasiswa telkom University mengajukan segala keluhan yang mereka dapatkan di lingkungan kampus.
baik itu fasilitas, lingkungan, pelecahan seksual, administrasi dll

fitur : 
1. Input keluhan 
2. Status keluhan
3. Login
4. Kategori keluhan 
5. Feedback
